Detailed Description

template<typename T, size_t N>
class RTML_buffer< T, N >

RTML_buffer allows instrumented applications and monitors to share the the same buffer by splinting it into read and write operations.

Monitor uses RTML_reader, and RTML_writer is reserved for software modules under observation.
